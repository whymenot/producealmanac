The berkeley Produce Almanac.

How to build:
    in Eclipse, (we strongly suggest that you use eclipse to build, and run)
    import the top-most directory of this project (Producealmanac).
    start build the project.

How to run:
    You need either,
    	1) A running android emulator (preferably Nexus7).
	2) An android device (preferably Nexus7) is connected to the computer.
    and, run the application in eclipse.
